Tags & Atributos

Tags são os blocos de construção do HTML que definem elementos, enquanto atributos fornecem informações ou configurações adicionais para essas tags.

Saiba Mais W3Schools

Textos

Existem diversas tags para formatação e estrutura de texto, como h1 a h6 para títulos hierárquicos e p para parágrafos de conteúdo.

Saiba Mais W3Schools

Tipos Button

Os botões permitem interação do usuário e podem ser do tipo submit (enviar formulário), reset (limpar campos) ou button (ação genérica via script).

Saiba Mais w3Schools

CheckBox & Radio

O input Checkbox permite selecionar múltiplas opções de uma lista, enquanto o Radio permite selecionar apenas uma opção única de um grupo.

Saiba Mais w3Schools

Div & Span

A Div é um container de bloco usado para agrupar elementos maiores, e o Span é um container de linha usado para estilizar partes específicas de um texto.

Saiba Mais w3Schools

Formularios

A tag form cria uma seção interativa que agrupa elementos de entrada para coletar dados do usuário e enviá-los a um servidor.

Saiba Mais w3Schools

Links & Midia

A tag 'a' cria hiperlinks para navegação entre páginas, enquanto tags como img, audio e video são usadas para incorporar mídia rica ao documento.

Saiba Mais w3Schools

Listas Ordenadas & Não Ordenadas

Listas organizam itens de forma estruturada, podendo ser ordenadas (ol) com numeração ou não ordenadas (ul) com marcadores.

Saiba Mais w3Schools

Tabelas

Tabelas organizam dados em linhas e colunas, utilizando a tag table em conjunto com tr (linhas), th (cabeçalhos) e td (células de dados).

Saiba Mais w3Schools

Tipos Input

O elemento input é versátil e possui vários tipos como text, password, email, number e date para receber diferentes formatos de dados do usuário.

Saiba Mais w3Schools

Cores

Cores podem ser aplicadas a elementos HTML via CSS utilizando nomes de cores, códigos hexadecimais, valores RGB ou HSL.
